# SimplyMed
### AI-Powered Medication Management Platform

A healthcare API that helps non-English speaking patients understand prescriptions through OCR scanning, multilingual translation, and AI-powered simplification of medical jargon.

## Tech Stack
Node.js, Express, PostgreSQL, Google Cloud Vision, Google Translate, Gemini AI, JWT, Firebase
